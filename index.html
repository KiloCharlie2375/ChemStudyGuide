Nomenclature & Polyatomic Ions

  • Chemical Nomenclature: The system for naming chemical compounds.
  • Three main classes: Ionic compounds, molecular (covalent) compounds, acids.
  • Correct names reveal what elements/ions are present and how many.
  • Metal + Nonmetal (or polyatomic): Name the cation (positive, metal) first, then the anion (negative, nonmetal or polyatomic).
  • Monatomic anions end in -ide (e.g., Cl- = chloride).
  • For metals with more than one possible charge (usually transition metals), indicate the charge with Roman numerals: FeCl2 is iron(II) chloride.
  • Example: Na2SO4 is sodium sulfate; Cu(NO3)2 is copper(II) nitrate.
  • Nonmetal + Nonmetal: Use prefixes to indicate number of atoms (see table below). Second element ends in -ide.
  • Omit “mono-” on the first element.
  • Example: CO2 = carbon dioxide; N2O5 = dinitrogen pentoxide.

  • Binary Acids: Contain H + nonmetal. Name: hydro- + root + -ic acid.
    Example: HCl (aq) = hydrochloric acid.
  • Oxyacids: Contain H + polyatomic ion (usually containing oxygen).
    - If polyatomic ends in -ate: root + -ic acid.
    - If ends in -ite: root + -ous acid.
    Example: H2SO4 (sulfate → sulfuric acid), H2SO3 (sulfite → sulfurous acid).

Matter & Measurement

  • Matter: Anything with mass and volume.
  • Classification: Pure substance (element/compound) or mixture (homogeneous/heterogeneous).
  • Physical Properties: Color, melting point, boiling point, density, etc.
  • Chemical Properties: Reactivity, flammability, ability to rust, etc.
StateShapeVolumeCompressibilityParticles
SolidDefiniteDefiniteNoFixed, tightly packed
LiquidIndefiniteDefiniteLowClose, can flow
GasIndefiniteIndefiniteHighFar apart, random
  • Plasma: Ionized gas, rare on Earth, common in stars.
  • Solids: Incompressible, expand slightly when heated.
  • Liquids: Not compressible, expand slightly, flow.
  • Gases: Compressible, expand to fill container, particles move fast.
  • Physical Change: No new substance (melting, dissolving, boiling, breaking).
  • Chemical Change: Produces new substance (rusting, burning, reacting with acid).
  • Clues for chemical change: Color change, gas production, precipitate, temperature change, light/energy released.

  • Molar Mass (MM): Mass of 1 mole of a substance (g/mol).
  • Molecular Weight: Another term for molar mass (used for molecules).
  • Calculate by adding atomic masses (from periodic table) for all atoms in formula.
  • Example: MM of H2O: (2×1.01) + 16.00 = 18.02 g/mol
  • Units: g/mol for both elements and compounds.
  • Density: Density = mass / volume, units: g/mL or g/cm³.
  • Percent Composition: (mass of element / total mass) × 100%
  • Percent Error: |measured - true| / true × 100%
Example: Density of a 12.5 g metal block, volume 2.5 mL?
Density = 12.5 g / 2.5 mL = 5.0 g/mL
Example: % composition of C in CO₂ (MM = 44 g/mol, C = 12 g/mol)?
(12 / 44) × 100% = 27.3%
Example: % error if measured 9.5g, true value 10g?
|9.5 - 10| / 10 × 100% = 5%

    Solution

  2. If a compound contains 7.20 g C and 1.20 g H, what is the % composition of H?
    Solution
    (1.20 / (7.20 + 1.20)) × 100% = 14.3%
  3. How many significant figures in 0.005400?
    Solution
    4 sig figs
  4. What is the percent error if you measure 48g, but the true mass is 50g?
    Solution
    |48 - 50| / 50 × 100 = 4%
  • Extensive Property: Depends on amount (mass, volume)
  • Intensive Property: Independent of amount (density, color, melting point)
  • Physical Property: Observed without changing composition
  • Chemical Property: Only observed by changing composition
  • Substance: Pure form of matter (element/compound)
  • Mixture: Physical blend of 2+ substances
  • Homogeneous: Same throughout (solution)
  • Heterogeneous: Not uniform throughout
  • Accuracy: Closeness to true value
  • Precision: Repeatability of measurement
  • SI Unit: International standard unit
  • Density: Mass per volume
  • Molar Mass: Mass per mole (g/mol)
  • Misconception: Mass and weight are the same. Fact: Mass is amount of matter; weight is gravity’s force on mass.
  • Misconception: All zeros are significant. Fact: Learn the rules!
  • Tip: Practice reading instruments—always estimate 1 decimal beyond marks.
  • Tip: Use units in every calculation for full credit.
  • Tip: Memorize SI prefixes (kilo-, milli-, centi-, etc.) for conversions.

Atoms & The Periodic Table

  • Atom: Smallest unit of an element, retains chemical identity.
  • Proton (p⁺): Charge +1, mass ≈ 1 amu, in nucleus.
  • Neutron (n⁰): Charge 0, mass ≈ 1 amu, in nucleus.
  • Electron (e⁻): Charge –1, mass ≈ 0 amu, outside nucleus (electron cloud).
  • Atomic Number (Z): # protons, defines the element.
  • Mass Number (A): protons + neutrons.
  • Isotope: Same element, different # neutrons (ex: C-12, C-13, C-14).
  • Ion: Atom or group with a charge (gain/loss of electrons).
Example: Find protons, neutrons, electrons for 37Cl⁻
  • Protons: 17 (atomic number)
  • Neutrons: 37 – 17 = 20
  • Electrons: 18 (17 + 1 for charge)

  • Groups (columns): Similar properties, same # valence electrons.
  • Periods (rows): Same number of electron shells.
  • Main Families:
    • Group 1: Alkali Metals (very reactive, +1 charge)
    • Group 2: Alkaline Earth Metals (+2)
    • Group 17: Halogens (–1)
    • Group 18: Noble Gases (inert)
  • Metals, Nonmetals, Metalloids: Metals left/center, nonmetals right, metalloids stair-step.
  • Transition Metals: Variable charges, often colored compounds.
Tip: Group number = # valence electrons (for A groups)
Group 17 (halogens) = 7 valence electrons.
  • Atomic Radius: Increases ↓ group, decreases → period.
  • Ionization Energy: Decreases ↓ group, increases → period (harder to remove e–).
  • Electronegativity: Increases →, decreases ↓ (F = highest).
  • Electron Affinity: More negative →, less negative ↓ (except noble gases).
  • Ionic size: Cations smaller, anions larger than their neutral atoms.
Practice: Which is bigger, K or Ca?
K (potassium), as it’s further left.
Mnemonic: "F is the greediest!"
Fluorine has the highest electronegativity and electron affinity.
  • Order of Fill: 1s, 2s, 2p, 3s, 3p, 4s, 3d, 4p, etc.
  • Aufbau Principle: Fill lowest energy levels first.
  • Pauli Exclusion: Max 2 e– per orbital (opposite spins).
  • Hund’s Rule: Fill orbitals singly before pairing.
  • Noble Gas Notation: Use previous noble gas in [brackets] as shortcut.

  • Molecule: Two or more atoms covalently bonded (ex: H₂O, O₂).
  • Formula Unit: Simplest ratio of ions in ionic compound (ex: NaCl).
  • Representative Particle: Atom, ion, molecule, or formula unit depending on substance.
  • Resonance: More than one valid Lewis structure (ex: O₃, NO₃⁻).
  • Oxidation Number: Apparent charge assigned by rules; used to track e– in reactions.
  • Element: Pure substance of one type of atom.
  • Ion: Atom or molecule with charge (cation = +, anion = –).

Bonding & Structure

  • Chemical bonds hold atoms together to form compounds.
  • Main bond types: Ionic (transfer e–), Covalent (share e–), Metallic (“sea of e–”)
  • Intermolecular forces (IMFs): Attractions between molecules (not within)
Bond TypeFormed byPropertiesExamples
Ionic Transfer e– (metal + nonmetal) High mp/bp, solid, conducts when molten/aq, brittle NaCl, MgO
Covalent Share e– (nonmetals) Low mp/bp, may be gas/liquid/solid, don't conduct H2O, CO2
Metallic Metal atoms, “sea of e–” Conductive, malleable, ductile, shiny Fe, Cu, Al
Hydrogen Bond H with N, O, or F High bp/mp for molecular, important for water, DNA H2O, NH3
  • Shows valence e– as dots; lines for bonds.
  • How to draw:
    1. Count total valence electrons.
    2. Arrange atoms (central = least electronegative, H never center).
    3. Single bonds between center & outer atoms.
    4. Fill outer atoms’ octets, then central atom.
    5. Double/triple bonds if needed to complete octet.
  • Exceptions: H (2e–), Be (4e–), B (6e–), expanded octet for P, S, Xe, etc.

Gases & Gas Laws

  • Gas: State of matter with indefinite shape and volume, compressible, expands to fill container.
  • Kinetic Molecular Theory (KMT):
    • Gases made of tiny, fast-moving particles.
    • Most volume is empty space; particles move randomly, collide elastically.
    • Temperature (K) is proportional to average kinetic energy.
PropertySymbolCommon Units
PressurePatm, kPa, mmHg, torr, psi
VolumeVL, mL, cm³
TemperatureTK (must use Kelvin!)
Amountnmoles (mol)
  • STP (Standard Temp & Pressure): 0°C (273 K), 1 atm, 22.4 L/mol gas.
  • Unit conversions: 1 atm = 101.3 kPa = 760 mmHg = 14.7 psi
  • Boyle’s Law: P₁V₁ = P₂V₂ (T, n constant)
  • Charles’s Law: V₁/T₁ = V₂/T₂ (P, n constant; T in K!)
  • Gay-Lussac’s Law: P₁/T₁ = P₂/T₂ (V, n constant)
  • Combined Gas Law: P₁V₁/T₁ = P₂V₂/T₂ (n constant)
  • Avogadro’s Law: V₁/n₁ = V₂/n₂ (P, T constant)
  • Ideal Gas Law: PV = nRT (R = 0.0821 L·atm/mol·K or 8.31 kPa·L/mol·K)
  • Dalton’s Law: Ptotal = P₁ + P₂ + ...
  • Graham’s Law (diffusion/effusion): Rate₁/Rate₂ = √(M₂/M₁)
Mnemonic: “Can Good People Always Party Down?”
Charles, Gay-Lussac, Boyle, Avogadro, Dalton
  1. Boyle’s Law: A 2.0 L gas at 1.0 atm is compressed to 0.5 L. What is new pressure?
    Solution
    P₁V₁ = P₂V₂ ⇒ (1.0)(2.0) = P₂(0.5) ⇒ P₂ = 4.0 atm
  2. Charles’s Law: If 1.5 L of gas at 300 K is heated to 450 K at constant P, new volume?
    Solution
    V₂ = V₁ × (T₂/T₁) = 1.5 × (450/300) = 2.25 L
  3. Ideal Gas Law: What volume does 3.0 mol O₂ occupy at 2 atm, 273 K?
    Solution
    V = nRT/P = (3.0 × 0.0821 × 273) / 2 = 33.6 L
  4. Dalton’s Law: O₂ at 0.2 atm, N₂ at 0.7 atm, CO₂ at 0.1 atm. Total pressure?
    Solution
    0.2 + 0.7 + 0.1 = 1.0 atm
  5. Graham’s Law: Which diffuses faster, H₂ or O₂?
    Solution
    Rate ratio = √(32/2) = 4; H₂ diffuses 4x faster!
  • Molar Volume: At STP, 1 mol gas = 22.4 L.
  • Density of Gas (d): d = PM/RT (M = molar mass)
  • Finding molar mass from gas data: MM = dRT/P
Example: Density of CO₂ at STP (M = 44 g/mol)
d = (1 atm × 44 g/mol) / (0.0821 × 273 K) ≈ 1.96 g/L
  • Real Gases: Deviate from ideal at high P, low T (particles have volume & IMFs)
  • van der Waals Equation: (P + a(n/V)²)(V–nb) = nRT (corrects for non-ideality)
  • Root Mean Square Speed: urms = √(3RT/M)

    Answer
    T = 300 K; P = (0.2 × 0.0821 × 300) / 5.0 = 0.984 atm
  2. If 3.00 L of a gas at 2.00 atm is expanded to 8.00 L at constant temperature, what’s new pressure?
    Answer
    P₂ = P₁V₁/V₂ = 2 × 3 / 8 = 0.75 atm
  3. Which gas effuses faster: methane (CH₄, 16 g/mol) or sulfur hexafluoride (SF₆, 146 g/mol)?
    Answer
    CH₄ effuses about √(146/16) ≈ 3x faster
  4. What is the density of O₂ (MM = 32 g/mol) at 25°C and 1.5 atm?
    Answer
    T = 298K; d = (1.5 × 32) / (0.0821 × 298) ≈ 1.96 g/L

  • Pressure: Force per unit area
  • Volume: Space gas occupies
  • Temperature: Average kinetic energy, always use K
  • Mole: Amount of substance, 6.022×10²³ particles
  • Molar Volume: 22.4 L/mol at STP

  • Diffusion/Effusion: Spread of gas (diffusion); passage through tiny hole (effusion)
  • Ideal Gas: Follows PV=nRT at all T/P (real gases deviate)
  • STP: Standard Temp (0°C/273 K) & Pressure (1 atm)

Equilibrium & Kinetics

  • Dynamic Equilibrium: Forward and reverse reactions occur at equal rates; concentrations constant, not necessarily equal.
  • Reversible Reaction: Products can reform reactants (⇌).
  • Example: N₂ + 3H₂ ⇌ 2NH₃
        Reactants  ⇌  Products
         (→ rate = ← rate)
  • At equilibrium, system is stable but constantly changing at the molecular level.
  • Kc: For aA + bB ⇌ cC + dD, 
    Kc = [C]^c [D]^d / [A]^a [B]^b
  • Kp: For gases, same format, but use partial pressures.
  • Solids and pure liquids are omitted from K expressions.
  • Interpreting K:
    • K > 1: Products favored at equilibrium
    • K < 1: Reactants favored
Example: Write Kc for H₂ + I₂ ⇌ 2HI
Kc = [HI]² / ([H₂][I₂])
  • If a system at equilibrium is disturbed, it will shift to oppose the change.
         Stress Added
             |
    +--------+--------+--------+
    |        |        |        |
Conc ↑   Temp ↑   Pressure ↑
    |        |        |
Shifts   Endo/exo   To fewer
away    (see ↓)    moles gas
from    +ΔH: right
added   –ΔH: left
  • Concentration: System shifts away from added, toward removed.
  • Temperature: Exothermic (–ΔH): heat is product; endothermic (+ΔH): heat is reactant.
  • Pressure (gases): ↑P shifts toward fewer moles gas; ↓P toward more moles.
  • Catalyst: No effect on equilibrium position (just faster to reach equilibrium).
Example: Add H₂ to N₂ + 3H₂ ⇌ 2NH₃?
Shifts right (toward NH₃, uses up H₂)
Example: Increase T for exothermic reaction?
Shifts left (reactant side)
  • ICE: Initial, Change, Equilibrium.
  • Set up concentrations, use stoichiometry for changes, fill in for K.
      A + B ⇌ 2C

      |     | A   | B   | C   |
      |-----|-----|-----|-----|
      | I   | 1.0 | 2.0 | 0   |
      | C   | –x  | –x  | +2x |
      | E   |1–x  |2–x  | 2x  |
Example: Kc = 4, [A] = 1–x, [B] = 2–x, [C] = 2x
Kc = [C]² / ([A][B]) = (2x)² / ((1–x)(2–x)) = 4
Step-by-step: Solving for x
4 = 4x² / ((1–x)(2–x))
4(1–x)(2–x) = 4x²
(1–x)(2–x) = x²
...expand and solve for x!
  • Reaction Rate: Change in concentration per unit time.
  • Factors affecting rate:
    • ↑Concentration = faster
    • ↑Temperature = faster (more collisions, more energetic)
    • ↑Surface area = faster
    • Catalyst: Lowers activation energy, increases rate (not consumed)
  • Activation Energy (Ea): Minimum energy required for reaction
  • Collision Theory: Molecules must collide with correct orientation and enough energy.
Example: Why does a catalyst help?
Lowers Ea, so more collisions have enough energy to react.
Example: Why does powder react faster than a chunk?
